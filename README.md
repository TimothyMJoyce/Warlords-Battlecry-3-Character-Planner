# Warlords Battlecry III Character Planner

## Goal

Create a clear, easy-to-use character planner for Warlords Battlecry III that helps players build, compare, and validate heroes.

## Summary

The planner lets users choose a hero's race, class, level, stats, skills, and portrait, then shows the resulting build values in one interface. It supports saved builds, local hero imports, and desktop use while keeping the planner focused on character-building decisions.
